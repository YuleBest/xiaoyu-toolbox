## Features

- **Modern UI Design**: Built with Vuetify 3, featuring Material Design style with a clean and elegant interface.
- **Responsive Layout**: Adapts to both desktop and mobile devices for efficient use anytime, anywhere.
- **Ad-Free**: A pure user experience with no intrusive ads.
- **Privacy Protection**: Most tools run entirely on the frontend — your data is never uploaded to a server.

## Getting Started

You can [try it online](https://tool.yule.ink) or deploy it yourself.

### Deployment Guide

```bash
git clone https://github.com/YuleBest/xiaoyu-toolbox.git
cd xiaoyu-toolbox
pnpm install
pnpm run build  # Build the project
pnpm run dev    # Development mode
```
