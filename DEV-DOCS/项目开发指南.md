# 项目开发指南

本指南旨在规范项目开发流程，确保代码质量和开发效率。

1. 本项目使用 Shadcn UI Vue 作为 UI 框架，请遵循 Shadcn UI 的开发规范。

2. 本项目使用 `pnpm` 作为包管理器，请使用 `pnpm` 进行包管理。

3. 若要添加新工具，请先在 `src/config/tools.ts` 中添加新功能的配置，然后在 `src/pages/` 目录下创建新功能的组件模板，并在 `src/i18n/` 目录下创建新功能的国际化配置。

4. 若新工具需用到 API，请先在 `src/config/api.ts` 中添加新功能的 API 配置，然后在 `src/api/` 目录下创建新功能的 API 模板，最后在 `src/pages/` 目录下使用。

5. 若需新增 Shadcn 组建，请使用 `pnpm dlx shadcn@latest add <component-name>` 命令添加组件。

7. 本项目的主题色是 `blue-500`。

8. 编写完成代码后，使用 `pnpm run lint` 命令检查代码规范和正确性，不必运行 `pnpm run build`。

9. 外观风格请参考现有工具，禁止引入大型 UI 库。
