## 特性

- **现代化 UI 设计**：基于 Vuetify 3 构建，采用 Material Design 风格，界面简洁美观。
- **响应式布局**：适配桌面端和移动端，随时随地高效使用。
- **无广告**：纯净的使用体验，无任何干扰广告。
- **保护隐私**：大部分工具纯前端运行，数据不上传服务器。

## 开始使用

你可以 [在线使用](https://tool.yule.ink) 或自行部署

### 部署指南

```bash
git clone https://github.com/YuleBest/xiaoyu-toolbox.git
cd xiaoyu-toolbox
pnpm install
pnpm run build  # 构建项目
pnpm run dev    # 开发模式
```
