# 🌦️ 中央气象台气象预警 API 接口文档

本文档用于记录中央气象台官网（nmc.cn）的实时气象预警数据接口。该接口包含**预警列表获取（JSON）**与**预警详情解析（HTML）**两个部分。

## 1. 获取预警信息列表

用于分页、条件筛选全国或各省份最新的气象预警信息。

### 📡 请求信息

- **请求 URL:** https://www.nmc.cn/rest/findAlarm
- **HTTP 方法:** GET
- **内容类型 (Content-Type):** application/json

### 📥 请求参数 (Query Parameters)

| 参数名      | 类型    | 是否必填 | 示例值        | 说明                                                           |
| ----------- | ------- | -------- | ------------- | -------------------------------------------------------------- |
| pageNo      | Integer | 是       | 1             | 当前页码（从 1 开始）                                          |
| pageSize    | Integer | 是       | 20            | 每页返回的条数                                                 |
| province    | String  | 否       | 广东省        | 筛选省/直辖市/自治区（**需全称**，如：广东省、广西壮族自治区） |
| signaltype  | String  | 否       | 台风          | 预警类型（如：台风、暴雨、雷雨大风、寒潮等），留空代表不限     |
| signallevel | String  | 否       | 蓝色          | 预警级别（如：蓝色、黄色、橙色、红色），留空代表不限           |
| \_          | Long    | 否       | 1783186271005 | 毫秒级时间戳，用于防止浏览器缓存（非必传）                     |

> **⚠️ 注意：** 参数包含中文时，在实际发送网络请求前，需进行标准 **URL 编码 (UrlEncode)**。

### 📤 响应数据结构 (JSON)

```json
{
  "code": 0,
  "msg": "success",
  "data": {
    "page": {
      "pageNo": 1,
      "pageSize": 20,
      "count": 51,
      "prev": 1,
      "next": 2,
      "list": [
        {
          "alertid": "44082541600000_20260704214523",
          "issuetime": "2026/07/04 21:45",
          "title": "广东省湛江市徐闻县气象台发布台风蓝色预警信号",
          "url": "/publish/alarm/44082541600000_20260704214523.html",
          "pic": "https://image.nmc.cn/assets/img/alarm/p0001004.png"
        }
      ]
    }
  }
}
```

### 🗂️ 响应字段说明

| 字段路径         | 类型    | 说明                                                      |
| ---------------- | ------- | --------------------------------------------------------- |
| code             | Integer | 状态码，0 表示请求成功                                    |
| msg              | String  | 状态描述信息                                              |
| data.page.count  | Integer | 满足当前筛选条件的总数据条数                              |
| data.page.list   | Array   | 预警数据列表对象数组                                      |
| ...[i].alertid   | String  | 预警唯一标识 ID                                           |
| ...[i].issuetime | String  | 预警发布时间（格式：YYYY/MM/DD HH:mm）                    |
| ...[i].title     | String  | 预警完整标题文本                                          |
| ...[i].url       | String  | 预警详情页相对路径（需自行拼接主域名 https://www.nmc.cn） |
| ...[i].pic       | String  | 预警对应的图标图片 URL 地址                               |

## 2. 获取预警详细正文

由于详情页为服务器端渲染的静态 HTML 页面，无法通过 JSON 接口直接读取，需请求网页并使用 DOM 解析器（如 XPath）提取内容。

### 📡 请求信息

- **请求 URL:** https://www.nmc.cn{url} _(将列表接口中返回的 url 字段进行拼接)_
- **示例 URL:** https://www.nmc.cn/publish/alarm/45062141600000_20260705010157.html
- **HTTP 方法:** GET

### 📥 请求 Headers 建议

- 为保障请求稳定性，建议携带基础的 User-Agent。
- 解析响应内容时，**必须指定文本编码为 UTF-8**，否则抓取到的中文会产生乱码。

### 🎯 数据提取定位 (数据清洗)

页面内的具体预警详情文字和防御指南嵌入在 HTML 的固定标签内，推荐使用以下选择器进行精确抓取：

- **XPath 定位表达式:**

  ```xpath
  //*[@id="alarmtext"]/p/text()

  ```

- **CSS 选择器表达式:**

  ```css
  #alarmtext > p

  ```

### 📤 提取文本示例

解析后拿到的纯文本数据结构示例：

> "上思县气象台2026年7月5日0时58分将暴雨橙色预警信号提升为红色：受台风“美莎克”环流云系影响，过去2小时……请注意防范。"

## 💡 开发者备忘/技巧

1.  **频率限制：** 该接口属于公共开放数据，但高频请求仍有被目标服务器 WAF（防火墙）临时封禁 IP 的风险。批量抓取详情页时，建议在代码逻辑中加入 time.sleep(1) 进行延时。
2.  **空数据处理：** 当输入的省份名称不匹配（例如误输入“广东”而非“广东省”）或当前地区无灾害预警时，列表接口会返回 code: 0 且 list: [] 的空数组，代码中需做好常规的长度判空。
